HOW TO CREATE NEW USERS
========================

1. Using Django Admin Panel:
   - Login to http://127.0.0.1:3000/admin/
   - Navigate to Users section
   - Click "Add User"
   - Note: Passwords must be bcrypt hashed

2. Using create_test_user.py script:
   - cd python
   - python manage.py shell
   - exec(open('create_test_user.py').read())

3. Using API Registration Endpoint:
   - POST to http://127.0.0.1:3000/api/auth/register
   - Provide email, password, and other required fields
